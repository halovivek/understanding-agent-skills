# Chapter 3: Machine Learning Fundamentals

## Learning Objectives

After completing this chapter, you will be able to:

- Understand the core concepts of Machine Learning.
- Differentiate supervised, unsupervised, semi-supervised, and reinforcement learning.
- Explain the end-to-end machine learning lifecycle.
- Recognize common algorithms and their enterprise applications.
