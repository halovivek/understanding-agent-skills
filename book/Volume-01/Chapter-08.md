# Chapter 8: AI Agents

## Learning Objectives

After completing this chapter, you will be able to:

- Define AI agents and distinguish them from traditional applications.
- Understand agent components such as planning, reasoning, memory, and tool use.
- Compare reactive, deliberative, and autonomous agents.
- Build a simple agent architecture.
- Identify enterprise use cases for AI agents.
