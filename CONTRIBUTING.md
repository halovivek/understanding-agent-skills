# Contributing to Understanding Agent Skills

Thank you for your interest in contributing.

## How to Contribute

1. Fork the repository.
2. Create a feature branch.
3. Follow the writing and coding standards.
4. Submit a pull request.

## Documentation Standards

- Use clear Markdown headings.
- Include runnable code examples.
- Add architecture diagrams where appropriate.
- Cite authoritative references.
- Test all sample code.

## Code Standards

- Python: PEP 8
- JavaScript: ESLint recommended rules
- Java: Google Java Style
- C#: Microsoft coding conventions

## Commit Messages

Use conventional commits, for example:

- docs: add chapter on RAG
- feat: implement MCP GitHub server
- fix: correct LangGraph example

## Code of Conduct

Please be respectful, constructive, and collaborative in all interactions.
